# Screenshots for SA-Data-Heartbeat

This directory holds the screenshots referenced by the main README and the Splunkbase listing.

When updating screenshots, keep filenames stable so links don't break:
- `monitor.png` — main Monitor dashboard
- `settings.png` — Settings page
- `audit.png` — Audit history
- `onboarding.png` — Welcome banner on a fresh install
- `mobile.png` — Mobile responsive view at <768px

Recommended capture: 1440x900 desktop, 414x896 mobile.
